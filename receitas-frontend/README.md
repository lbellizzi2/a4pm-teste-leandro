# Receitas Frontend

## Como iniciar a aplicação

1. Instale as dependências:
   ```bash
   yarn
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   yarn dev
   ```

3. Acesse a aplicação no navegador no endereço indicado no terminal.
